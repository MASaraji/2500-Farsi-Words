# 2500-Farsi-Words

This repository contains a list of the 2,500 most frequently used words in Farsi (Persian). It's a simple and useful resource for:

- Language learners
- Developers
- Researchers

## 📂 Files

- `words.csv`: Contains the 2,500 words collected from wikipedia
